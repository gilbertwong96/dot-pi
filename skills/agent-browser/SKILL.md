# agent-browser

Load current CLI-bundled docs first:

```bash
agent-browser skills get core
```

For complex tasks:

```bash
agent-browser skills get core --full
agent-browser skills list
```

Setup:

```bash
npm install -g agent-browser
agent-browser install
```
